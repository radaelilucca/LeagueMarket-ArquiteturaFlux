## :rocket: Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- [ReactJS](https://reactjs.org)
- [Redux](https://redux.js.org/)
- [Redux-Saga](https://github.com/redux-saga/redux-saga)
- [JSON Server](https://github.com/typicode/json-server)

## 💻 Projeto

O LeagueMarket é um projeto que simula um e-commerce tendo o Carrinho como estado global sendo acessado por diversos componentes da aplicação.

<p>Desenvolvido no Bootcamp GoStack <a href="https://rocketseat.com.br/">Rocketseat</a>!</p>

## 👨‍🏫 Executando

Execute o comando yarn na pasta do projeto para baixar todas as depêndencias necesárias para rodar a aplicação.

<h4> Inicie o projeto em modo desenvolvimento com os seguintes comandos: </h4>

<h4> Front-end: </h4>

```
yarn start
```

<h4> Back-end: </h4>

```
json-server server.json -p 3333 -w
```
